# [Nome do Projeto/Area]

Copie esta pasta e renomeie com o nome do seu projeto ou area.

## Estrutura

- `docs/` — Documentos relevantes: relatorios, apresentacoes, manuais, atas
- `dados/` — Planilhas, CSVs, exports de sistemas, numeros
- `referencias/` — Material de apoio: artigos, benchmarks, templates

## Como usar

1. Jogue seus arquivos nas pastas correspondentes
2. Rode `/contexto` para a IA ler tudo e gerar o CLAUDE.md
3. Comece a trabalhar — a IA ja vai conhecer o contexto

## Dicas

- Quanto mais dados voce colocar, melhor a IA trabalha
- Atas de reunioes sao ouro — a IA entende contexto, prioridades e decisoes
- Atualize os dados periodicamente e rode `/contexto` de novo
