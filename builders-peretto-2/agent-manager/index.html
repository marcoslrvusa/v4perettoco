📖 Como Usar o Agent Manager

1. A Inteligência do Sistema (Contexto Duplo)

A IA deste sistema não é genérica. Ela opera usando duas mentes conectadas:

  • 🟢 Motor Global (Manual Peretto): Fica sempre ligado. Garante que todos os agentes pensem e respondam seguindo os processos da V4 (SPICED, FCA, Quality Check).
  • 🔵 Motor Específico (Cliente): Ligado apenas quando você seleciona um cliente no menu esquerdo. Injeta a realidade da conta para que as respostas (como textos de copy) sejam personalizadas para aquele negócio.

2. Executando Rituais V4

Na aba lateral direita "Tarefas", você encontra os botões de execução de rituais com um clique:

📋 Quality Check Auditoria sistêmica da conformidade das squads (Atas, Rituais). Use com o Agente Coordenador.
🔥 Sabatina de Plano Stress-test para mapear dependências antes da execução. Cole o plano e inicie a sabatina.
👔 Account Planning Ritual de Onboarding estratégico (Situation, Pain, Impact, Critical Event, Decision). Use com o Account Manager.

3. Os Agentes Especialistas

Cada agente na barra esquerda tem um prompt secreto de sistema alinhado com a franquia:

  • Copywriter V4: Constrói jornadas com PAS, AIDA e checklist de copy V4.
  • CSM V4: Focado no LTV e retenção. Identifica sinais de fumaça (churn).
  • Analista V4mos: Puxa e cruza dados das integrações Meta/Google para diagnosticar funis (ROAS/CPA).
  • Onboarding (Setup): Cria as pastas e estrutura de NotebookLM para novos clientes automaticamente.
