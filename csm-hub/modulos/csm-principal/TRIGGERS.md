# TRIGGERS — CSM Principal V1

| Você diz... | O agente faz... |
|---|---|
| "Inicie o setup da unidade [nome]" | Guia setup completo passo a passo |
| "Flag de [tipo] no cliente [X]" | Triagem + decisão documentada |
| "Acione [área] para o cliente [X]" | Template de acionamento completo |
| "Qual a saúde da minha carteira?" | Resumo por status com flags abertas |
| "Prepare o QBR do cliente [X]" | Estrutura completa do QBR |
| "Feche o loop da flag [X]" | Checklist de fechamento + registros |
| "Há oportunidade de expansão em [X]?" | Análise de expansão + proposta |

---

## Situações que ativam o agente

- Flag automática chegou por email — precisa decidir o que fazer
- Fim de quarter — QBR de todos os clientes
- Cliente sinalizou insatisfação fora da pesquisa formal
- Área acionada reportou que resolveu — precisa validar e fechar
- Reunião com cliente amanhã — precisa de briefing de saúde
- Nova unidade implementando o CSM — setup inicial
