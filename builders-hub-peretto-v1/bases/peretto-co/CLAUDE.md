# Peretto e Co — Hub da Unidade

Bem-vindo ao hub operacional da unidade franqueada **Peretto e Co**. Este espaco foi customizado para refletir nossa realidade de operacao, focada em escala e integracao de dados de performance (Google Ads, Meta Ads).

## Estrutura da Unidade

- `bases/peretto-co/clientes/` — Nossos clientes locais.
- `bases/peretto-co/processos/` — POPs e Metodologias Peretto.
- `bases/peretto-co/docs/` — OKRs da unidade e planejamento estrategico.

## Padroes de Operacao

1. **Integracao Google Ads**: Todo cliente deve ter as credenciais preenchidas no `.env` para permitir auditorias automatizadas.
2. **Escalabilidade**: Use a skill `/novo-cliente-peretto` (em desenvolvimento) para garantir que a estrutura venha com todos os nossos docs base.
3. **Frequencia**: Check-ins semanais sao baseados nos dados extraidos via API.

## Comandos Rapidos

- `/contexto` — Rode dentro da pasta do cliente para atualizar a visao da IA.
- `/sabatina` — Use antes de apresentar qualquer plano ao franqueado.
- `/onboarding` — Se o ambiente quebrar.
